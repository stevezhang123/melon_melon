# Melon Melon / 种瓜得瓜 1.0.0

## Supported Versions

- Minecraft: 1.21.1
- Loader: NeoForge
- Release Type: Release

## Features

- Adds ?Tutu Soil?, a special configurable soil block.
- Allows configured items and blocks to be planted directly.
- Supports upright item rendering and block/item fallback rendering.
- Supports Create namespace wildcard planting via `create:*`.
- Supports staged planting whitelist.
- Supports seed-retaining right-click harvest.
- Adds ?Feifei? as a growth booster.
- Right-click harvest drops 1–4 outputs with weighted chances.
- Adds survival crafting recipes for ?Tutu Soil? and ?Feifei?.
- Includes bilingual Chinese-English configuration comments.

## Recipes / 合成配方

- ?Tutu Soil? is crafted with diamonds, fern, gilded blackstone, netherrack, and dirt. The recipe outputs 4 blocks and costs 4 diamonds, averaging 1 diamond per soil block.
- ?Feifei? is crafted with sculk, bone meal, and moss blocks. Each craft outputs 16 ?Feifei?.

- ？肥肥？需要幽匿块、骨粉和苔藓块合成。每次输出 16 个。
